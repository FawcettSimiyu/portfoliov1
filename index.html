Project 2: Networking

Objective: Set up and configure a network on Linux, ensuring robust security and efficient communication between devices.

Tools Used: Linux Mint, iproute2, iptables, Bash scripting.

Project Goals:

  • Configure network settings including static IP, DNS, and routing tables.
  • Implement firewall rules to enhance network security.
  • Document the setup for future reference and troubleshooting.

Methodology:

  • Used `ip route` to configure routing tables for multiple network interfaces.
  • Set up firewall rules using `iptables` to restrict unauthorized access.
  • Documented network setup including IP addresses, DNS, and routing configuration.

Outcomes:

  • Established a reliable and secure network environment.
  • Created detailed documentation for network management and troubleshooting.

Project 3: Campaign Contribution Analysis Using AWK Scripting

Objective: Analyze campaign contributions using AWK scripting to derive insights into donor behavior and contribution patterns.

Tools Used: AWK, Bash, Command-line interface, CSV data files.

Project Goals:

  • Analyze large datasets to extract insights on campaign contributions.
  • Identify trends and donor behavior over time.
  • Provide actionable recommendations based on the data analysis.

Methodology:

  • Used AWK for data manipulation and analysis on large CSV files.
  • Filtered contributions by candidate, time period, and donor.
  • Calculated aggregate contributions to identify top donors.

Outcomes:

  • Extracted trends in contributions and identified key donors.
  • Enhanced my data analysis and scripting skills using AWK.
